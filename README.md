# Tools

A collection of local-first productivity tools built with FastAPI backends and React/Vite/Tailwind frontends.

## Calendar & Email Summary

AI-powered summaries of your Gmail inbox and Google Calendar. Pick a time period (day, week, month, quarter), choose emails or calendar (or both), and get a concise summary with highlights, themes, and action items generated by Claude or OpenAI.

**Key features:**
- Summarize inbox emails or calendar events over any time period
- Scheduled jobs — set up cron-like recurring summaries (hourly, daily, weekdays, weekly, monthly)
- Each job is fully customizable: combine email and calendar tasks with independent time periods and directions
- Reports are saved and browsable in the web UI
- Optional macOS notifications (banner or persistent popup dialog) when a scheduled report completes
- Multiple AI providers: Anthropic API, OpenAI API, Claude Code CLI, Codex CLI

**Stack:** FastAPI, React, TypeScript, Tailwind CSS, APScheduler, Google Gmail & Calendar APIs

[Full documentation →](calendar-email-summary/README.md)

## Finance Tracker

Local-first personal finance dashboard. Aggregate accounts from Plaid (12,000+ institutions), SimpleFIN Bridge, or manual entry with CSV import — then see consolidated net worth, spending breakdowns, income analysis, and goal-based financial plans.

**Key features:**
- Multi-source aggregation: Plaid, SimpleFIN, and manual accounts with CSV import
- Net worth with asset/liability breakdown (Cash, Investments, Retirement, Credit Cards, Loans, …)
- Dynamic spending categories with user CRUD — create, rename, and delete categories from the UI
- Frequency-based cost projections (weekly/biweekly/monthly/quarterly/annual) per merchant
- Account ignore: exclude accounts from analytics while keeping them synced; transfers to ignored accounts count as spending via counterpart-based detection
- Income analysis over configurable 30–365 day windows with per-source deposit history
- Dashboard with monthly/quarterly/annual projections for income, spending, and cash flow
- Goals & financial planning with feasibility ratings

**Stack:** FastAPI, plaid-python, React, TypeScript, Tailwind CSS, react-plaid-link

[Full documentation →](finance-tracker/README.md)

## Gmail Manager

A local-only tool for burning down a bloated Gmail inbox. Combines a dashboard for manual triage with a long-lived CLI cleanup agent (Claude Code or Codex) that proposes bulk actions — delete, block, unsubscribe — for one-click review.

**Key features:**
- Dashboard showing top senders, storage usage, and unsubscribe candidates
- One-click delete, block, and unsubscribe actions with bulk support
- Agent-driven cleanup: a CLI agent runs in a loop, discovers high-volume senders and stale mail, and posts recommendations to the dashboard for approval
- Download emails as `.eml` + attachments in a ZIP before deleting
- Configurable rules: protected senders/keywords, minimum age, approval requirements

**Stack:** FastAPI, React, TypeScript, Tailwind CSS, Google Gmail & Drive APIs

[Full documentation →](gmail-manager/README.md)

## Quick Start

Each tool has its own `start.sh` that installs dependencies and launches both backend and frontend:

```bash
# Calendar & Email Summary
cd calendar-email-summary
./start.sh
# Open http://localhost:5174

# Gmail Manager
cd gmail-manager
./start.sh
# Open http://localhost:5173

# Finance Tracker
cd finance-tracker
./start.sh
# Open http://localhost:5175
```

The Google-powered tools require Google Cloud OAuth 2.0 credentials (Gmail API + Calendar API or Drive API enabled). Finance Tracker supports Plaid, SimpleFIN, and manual account entry — see its README for setup details.
